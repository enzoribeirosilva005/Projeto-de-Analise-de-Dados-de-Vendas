## 🚀 Sobre o Projeto
Este projeto tem como objetivo realizar uma análise completa de dados de vendas, utilizando bibliotecas do Python para transformar dados brutos em informações estratégicas.

A aplicação permite:
- Filtrar e organizar dados
- Gerar gráficos dinâmicos
- Visualizar informações através de uma interface gráfica simples e intuitiva

💡 **Proposta:**
Utilizar múltiplas bibliotecas Python para fornecer insights que auxiliem na tomada de decisão em um mercado competitivo.

## 🎯 Objetivos
- 📈 Analisar dados de vendas por meio de gráficos
- 🖥️ Criar uma interface gráfica intuitiva
- 🔎 Aplicar filtros para melhor organização dos dados

## 🛠️ Tecnologias Utilizadas
- Python
- Pandas → Manipulação e análise de dados
- Matplotlib → Geração de gráficos
- Tkinter → Interface gráfica

## 📂 Estrutura dos Dados
O projeto utiliza arquivos `.csv` como base de dados:
- vendas.csv
- produtos.csv
- (outros arquivos auxiliares)

Esses arquivos são carregados e processados para gerar análises visuais.

## 🔍 Etapas do Desenvolvimento
### 📌 1. Tratamento de Dados com Pandas
- Importação dos arquivos `.csv`
- Limpeza de dados:
  - Remoção de espaços em branco
  - Padronização de colunas
- Conversão de datas para datetime
- Criação de novas colunas
- Junção de tabelas
- Armazenamento dos dados processados

📊 **Resultado:** Dados organizados e prontos para análise

### 📊 2. Visualização com Matplotlib
- Criação de gráficos de linha
- Análise da evolução das vendas ao longo do tempo
- Customização:
  - Tamanho
  - Cores
  - Rótulos

📈 **Resultado:** Visualização clara e objetiva das vendas

### 🖥️ 3. Interface Gráfica com Tkinter
Integração completa entre:
- Pandas
- Matplotlib
- Tkinter

**Funcionalidades:**
- 📂 Seleção de arquivos (filedialog)
- ⚠️ Alertas e mensagens (messagebox)
- 📊 Exibição de gráficos na interface
- 🔄 Processamento automático dos dados

**Interface:**
- Tela inicial com botões
- Carregamento de arquivos
- Exibição de gráficos interativos

## 📊 Funcionalidades do Sistema
- Importação de dados de vendas
- Organização automática dos dados
- Cálculo do valor total de vendas
- Geração de múltiplos gráficos
- Interface amigável para o usuário

## 📈 Resultados
O sistema entrega:
- Dados organizados e estruturados

- Melhor compreensão do desempenho de vendas
